**MuslimahBride** adalah halaman arahan e-commerce premium yang dirancang khusus untuk butik kerudung dan gaun pengantin muslimah mewah. Proyek ini menampilkan antarmuka modern dan elegan dengan pengalaman berbelanja yang lancar, didukung oleh sistem penata gaya AI simulasi.

## Fitur Utama

- **Galeri Produk Elegan:** Tata letak grid yang memukau secara visual untuk menampilkan koleksi eksklusif pernikahan.
- **Penata Gaya AI (Alya):** Chatbot terintegrasi (didukung oleh OpenAI/OpenRouter) yang membantu pengguna dengan saran gaya dan pengecekan stok barang.
- **Sistem Keranjang Belanja & Wishlist:** Keranjang belanja yang berfungsi dengan manajemen state (disimpan di LocalStorage), mendukung penambahan dan penghapusan item. Serta fitur Wishlist terintegrasi.
- **Pencarian Produk:** Fitur pencarian langsung (live search) untuk memudahkan pengguna menemukan produk yang mereka inginkan.
- **Desain Responsif:** Memastikan pengalaman pengguna yang optimal di berbagai ukuran layar, mulai dari perangkat seluler hingga desktop.
- **Performa Cepat:** Dibangun menggunakan Vite untuk Hot Module Replacement (HMR) yang sangat cepat dan hasil build produksi yang optimal.
- **Sistem Manajemen Konten (CMS):** Dasbor admin terintegrasi untuk mengelola data produk (dilengkapi dengan panel analitik dan manajemen pesanan dalam pengembangan).

## Teknologi yang Digunakan

- **Frontend:** React 19, TypeScript
- **Styling:** Tailwind CSS v4
- **Animasi:** Framer Motion
- **Ikon:** Lucide React
- **Build Tool:** Vite
- **Integrasi AI:** OpenAI SDK (OpenRouter)

## Instalasi dan Pengaturan

1. **Clone repositori ini**
   ```bash
   git clone https://github.com/ptramichsthg/Muslimah-Bride.git
   cd Muslimah-Bride
   ```

2. **Instal dependensi**
   ```bash
   npm install
   ```

3. **Atur Variabel Lingkungan**
   Buat file `.env` di direktori utama dan tambahkan kredensial API Anda:
   ```env
   VITE_OPENROUTER_API_KEY=kunci_api_anda_disini
   ```

4. **Jalankan server pengembangan**
   ```bash
   npm run dev
   ```
